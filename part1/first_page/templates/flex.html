Flex box

The best info in the Internet about flex boxes

Simple page used to be like:

as far as all are blocks, they are one by one.

to start working with flex box, we have to change display property to flex

Flex direction

flex container have two axes:

flex-direction helps us to change the axes

also flex-direction can be column-reverse

Justify Content

justify content is responsible for position across the main axis

It can be:

  1. flex-start;
  2. flex-end;
  3. center;
  4. space-between;
  5. space-around.

Align Items

align items works with Cross axis.

It can be:

  1. flex-start;
  2. flex-end;
  3. center;
  4. stretch;
  5. baseline.
